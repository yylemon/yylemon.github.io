开启root账户的SSH登录

2017-11-20
install

由于我习惯用root账户了，实验室新来的服务器，给的一个用户名dell的登陆方式，当然是可以登录后切换的，但也设置一下允许直接使用root登录。
方法主要借鉴于：
https://jingyan.baidu.com/article/d2b1d102abf7975c7f37d450.html

1 安装openssh：
xxx@ubuntu:~$ sudo apt-get install openssh-server
2 修改root密码
xxx@ubuntu:~$ sudo passwd root
3 切换为root用户，修改config：
xxx@ubuntu:~$ su - root   #切换到root账户
Password:                 #输入第二步修改的root密码
root@ubuntu:~# vim /etc/ssh/sshd_config

找到下面这段代码，注释掉“PermitRootLogin prohibit-password”，添加“PermitRootLogin yes” 

# Authentication: 
LoginGraceTime 120 
#PermitRootLogin prohibit-password
PermitRootLogin yes 
StrictModes yes
4 重启ssh服务
root@ubuntu:~# sudo service ssh restart 
root@ubuntu:~#

完毕，可以使用root账户直接ssh登录服务器
