java+tomcat+mysql环境配置

2017-11-12
install, java, mysql, tomcat

趁还有学生优惠的时候，购了一年的阿里云服务器，记录一下基本javaweb的配置。

==系统：ubuntu1 16.04==

1.java环境

(1)在根目录下新建download和java文件夹 

$ mkdir /download
$ mkdir /usr/java

(2)下载jdk

下载jdk至/download,jdk地址：
http://www.oracle.com/technetwork/java/javase/downloads/index.html

(3)将jdk解压至/usr/java 

$ cd /download
$ tar zxvf jdk-8u144-linux-x64.tar.gz -C /usr/java

修改名称： 

$ mv jdk1.8.0_25 jdk1.8

(4)配置环境变量

$ vim /etc/profile

在文件末尾加入： 

$ export JAVA_HOME=/usr/java/jdk1.8 
$ export JRE_HOME=${JAVA_HOME}/jre
$ export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib  
$ export PATH=${JAVA_HOME}/bin:$PATH

使环境变量生效： 

$ source /etc/profile

查看java环境 

$ java -version
2.安装tomcat

(1)下载tomcat至/download
https://tomcat.apache.org/download-90.cgi

解压至/usr/java: 

$ cd /download
$ tar zxvf apache-tomcat-9.0.1.tar.gz -C /usr/java

修改名称： 

$ mv apache-tomcat-9.0.1 tomcat9

(2)配置环境变量

$ vim /etc/profile

在文件末尾加入：

$ export CATALINA_HOME=/usr/java/tomcat9

使环境变量生效： 

$ source /etc/profile
(3)启动tomcat
$ cd /usr/java/tomcat9
$ ./bin/startup.sh

打开浏览器：http://服务器ip:8080/
可以看到如下界面：

3.安装mysql

(1)安装 

$ apt-get update
$ apt-get install mysql-server mysql-client libmysqlclient-dev

在此安装过程中会让你输入root用户(管理MySQL数据库用户，非Linux系统用户)密码，按照要求输入即可。

(2)测试安装是否成功 

$ sudo netstat -tap | grep mysql
$ mysql -u root -p
$ service mysql start
$ service mysql stop
$ service mysql restart

(3)修改配置文件，方便远程连接

$ vim /etc/mysql/mysql.conf.d/mysqld.cnf

将“bind-address = 127.0.0.1”这一行注释掉。 

#bind-address = 127.0.0.1
